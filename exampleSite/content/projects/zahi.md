## نظرة عامة

Zahi هو قالب Hugo مصمم لدعم المحتوى العربي بشكل أساسي مع واجهة مستخدم بسيطة ونظيفة.

![Zahi Hugo Theme](https://raw.githubusercontent.com/mohamedelhefni/zahi/main/images/screenshot.png)

## المميزات

- **أداء متميز**: سرعة تحميل عالية
- **فهرس المحتوى**: جدول محتويات منظم
- **المقالات المثبتة**: تثبيت المقالات المهمة
- **الوسوم**: تصنيف المحتوى بسهولة
- **دعم Disqus**: قسم التعليقات
- **التصفح**: تقسيم المحتوى إلى صفحات
- **تمييز الأكواد**: عرض الأكواد بوضوح
- **تصميم متجاوب**: يعمل على جميع الشاشات
- **الوضع الداكن/الفاتح**: حسب تفضيلات المستخدم

## ميزات الحديقة الرقمية

- **روابط Wiki**: دعم [[Wiki Style]]
- **خريطة المعرفة**: رسم بياني تفاعلي
- **الروابط الخلفية**: عرض الصفحات المرتبطة
- **السلاسل التعليمية**: تنظيم المقالات

## التطوير

```bash
# تشغيل الخادم
hugo server

# مراقبة Tailwind
npx tailwindcss -i ./static/css/style.css -o ./static/css/output.css --watch
```
