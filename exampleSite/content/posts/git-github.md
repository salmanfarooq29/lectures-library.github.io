## نظام التحكم في الإصدارات

Git هو نظام للتحكم في الإصدارات يساعد المبرمجين على تتبع التغييرات في الكود وإدارة المشاريع.

### لماذا Git؟

- تتبع جميع التغييرات
- العمل الجماعي بسهولة
- الرجوع لأي نسخة سابقة

### الأوامر الأساسية

```bash
git init
git add .
git commit -m "رسالة"
git push
```

### GitHub

GitHub هو منصة لاستضافة مشاريع Git وتتيح:

- مشاركة الكود
- التعاون مع فريق
- إدارة المشاريع
